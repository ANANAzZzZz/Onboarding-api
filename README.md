# Дипломный проект "Онбординг сотрудника"

Backend часть приложения дипломного проекта "Онборлинг сотрудника"

## Описание

Данный проект представляет собой серверную часть мобильного приложения для первичной адаптации новых сотрудников, разработанную в рамках выпускной квалификационной работы.

Система обеспечивает автоматизацию процессов онбординга и интеграцию с мобильным клиентом через REST API. Решение ориентировано на образовательные учреждения, но может быть адаптировано под другие организации.

## Технологический стек

- **Java 17**
- **Spring Boot**
- **PostgreSQL**
- **Redis**
- **Docker Compose**
- **Prometheus + Grafana**
- **Apache JMeter**
- **Swagger/OpenAPI**

## Как запустить

1. Убедитесь, что установлен Docker и Docker Compose
2. Клонируйте репозиторий
3. Запустите команды для старта окружения:

```bash
cd .\scripts\
docker-compose up --build
```
4. Запустите само приложение
